# 视觉方向和资产 Prompt

## 目标

卡通 2.5D 牧场棋盘。视觉要像精致移动休闲游戏，不像停车场换皮，也不像默认 canvas demo。

## 关键词

- isometric 2.5D
- cozy pasture puzzle
- cute sheep
- toy-like shapes
- soft shadows
- bright readable mobile game
- clean silhouettes

## 禁止方向

- 不要真实 3D 引擎感。
- 不要低质 Unity primitive 感。
- 不要停车场、车库、公路视觉。
- 不要把羊画成一个白球。
- 不要复杂纹理导致小屏看不清。

## 羊 Sprite Prompt

```text
A cute isometric 2.5D sheep sprite sheet for a mobile puzzle game, four directions north east south west, rounded fluffy cream wool, warm tan face, tiny ears, short dark legs, friendly eyes, clean silhouette, toy-like casual game style, soft shadow, transparent background, consistent feet-center anchor, idle and running frames, readable at 96 pixels high, no realistic fur, no text.
```

## 羊仓 Sprite Prompt

```text
A charming isometric 2.5D sheep barn for a mobile puzzle game, red wooden walls, rounded blue-gray roof, open dark doorway facing the board, small sheep icon sign, hay details, soft shadow, cozy bright colors, transparent background, readable on mobile, no text, no realistic grime.
```

## 棋盘/装饰 Prompt

```text
An isometric 2.5D cozy pasture puzzle board, grass diamond tiles, raised rounded edges, alternating green tones, tiny flowers, fence pieces, hay bales, soft shadows, bright daylight, mobile casual puzzle game style, transparent background elements, no cars, no road, no parking lot.
```

## 下一阶段视觉验收

- 羊在 4 个方向都能看出脸/身体朝向。
- 羊跑动时脚步或身体弹跳明显。
- 羊仓入口方向清楚，玩家知道目标。
- 错误点击有红色落点反馈。
- 棋盘格和实体不重叠，不影响点击判断。
