# 产品体验与留存功能落地记录

更新日期：2026-06-11

## 已落地

- 新手引导：首次进入第 1 关时标亮安全小羊，并用贴边浮层说明核心操作。
- 新手保护：前 20 关误点、点障碍、点被挡小羊不直接失败，改为记录失误、扣星并显示落点或阻挡路线。
- 高阶压力：第 21 关后保留失败页，避免后期玩法失去挑战。
- 阻挡反馈：被羊或障碍挡住时，Canvas 会画出路径提示和阻挡终点。
- 免费提示：HUD 与失败页救援入口可标亮当前安全小羊；当前版本不接广告 SDK。
- 撤销：局内可撤销上一步，撤销会把本关最高星级限制到 2 星。
- 星级：按失误和辅助使用情况结算 1-3 星，并在关卡选择中显示历史最高星级。
- 每日挑战：首页提供每日挑战入口，按本地日期映射到前 40 关，完成后记录每日 streak；每日挑战不跳过主线解锁进度。
- 成就摘要：首页展示成就完成数、总星数和最佳连胜。
- 关卡曲线：前 20 关从 4 只羊平滑提升到 25 只羊；第 21 关继续阶段跃迁，后期仍达到 220 只羊和 25 个障碍。

## 本轮验证

- `npm run levels:validate`：200 个关卡均可解，最大棋盘 16x16，最大羊数 220，最大障碍数 25。
- `npm run build`：TypeScript 与 Vite 生产构建通过。
